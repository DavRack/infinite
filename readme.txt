¿Cómo iniciar la aplicacion?

	Asegurarse de tener NetBeans 8.2 con los complementos web necesarios

	En el archivo /models/txt.java cambiar el valor de la variable file_name por la ruta al archivo que se desea usar como base de datos

	Con el proyecto abierto en NetBeans oprimir la tecla F6
